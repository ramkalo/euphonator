## Guide

### How To:
Chord Abbreviation -> "Maj"
Semitones relative from previous note -> "0 4 3"

ex. 1} The chord "A Major" is calculated as:
A + 0 = A
A + 4 semitones = C#
C# + 3 semitones = E

ex. 2} The chord "B minor first inversion" is calculated as: 
B + 3 = D
D + 4 = F#
F #

# Ram Kalo Pentatonic Index

Full list of all distinct pentatonic scale shapes that produce no modal synonyms

How to read these:
pick a note, lets say C, then pick an index, lets say #31 -> 1214. Start at C then move up 1 semitone to C#, then 2 semitones to D#, then 1 semitone to E, then 4 semitones to G# so the pentatonic scale that results is = C-C#-D#-E-G#.
